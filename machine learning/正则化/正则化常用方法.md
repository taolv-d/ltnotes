
[[L1 L2 正则化]]
### 🔧 参数范数约束

这类方法直接对模型的权重参数施加惩罚，L1/L2就属于此类。另一个重要的变体是：

- **Elastic Net（弹性网络）**：结合 L1 和 L2 的惩罚项。它既能像 L1 那样产生**稀疏性**（让部分权重变为0），又保留了 L2 的**稳定性**。特别适合处理特征数量远多于样本量，或特征间高度相关的场景。


### 🧠 神经网络结构优化

这是专门为深度学习设计的主流方法：

- **Dropout**：训练时，随机“丢弃”一部分神经元（让其不工作）。这迫使网络学习更鲁棒的特征，防止神经元之间产生复杂的共适应关系，是防止过拟合的利器 [](https://hub.baai.ac.cn/knowledge-tree/c5638c5e-b697-4f1e-9134-49826df1b1d8)。
    
- **Batch Normalization（批归一化）**：将每一层网络的输入分布拉回标准正态分布。虽然主要作用是加速训练，但同时也因为引入了轻微的噪声，带来了一定的**正则化副作用** [](https://hub.baai.ac.cn/knowledge-tree/c5638c5e-b697-4f1e-9134-49826df1b1d8)。
    

### 🖼️ 数据增强

- **数据增强 (Data Augmentation)**：不修改模型，而是通过随机旋转、裁剪、翻转、变色等方式**扩充训练集**。模型见的“世面”多了，自然就不容易“死记硬背”原始图片，泛化能力也就更强了 [](https://www.ultralytics.com/zh/glossary/regularization)。
    

### ⏱️ 训练过程干预

- **Early Stopping（早停法）**：监控模型在**验证集**上的表现。一旦发现验证误差不再下降甚至开始反弹，就立即终止训练。这能防止模型过度学习训练集中的噪声，是操作最简单、却最有效的方法之一 [](https://hub.baai.ac.cn/knowledge-tree/c5638c5e-b697-4f1e-9134-49826df1b1d8)。
    

### 🏷️ 标签处理

- **Label Smoothing（标签平滑）**：将真实标签（通常是独热编码的 0 和 1）变得“软”一点，例如把 `[0, 1]` 变成 `[0.05, 0.95]`。这能**防止模型对预测过于自信**，避免它为了追求极端的 0 或 1 而走向过拟
