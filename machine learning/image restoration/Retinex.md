# Retinex 理论
Retinex 直接翻译是 **视网膜皮层**，一般直接叫**Retinex 理论**

Retinex 理论：**你看到的图像 = 物体的“本色”（反射分量） × 环境的“光线”（光照分量）**，即人感受到的颜色并不是物体本身固定的属性，而是它的反射能力和当时光照共同作用的结果：
$$S=R*I$$
其中，$S$是最终得信号，$R$是反射分量(reflection)，$I$是光照(illumination)。乘法是逐像素相乘

Retinex 的目标是在一张图像中分离$I$，还原出真实$R$（让机器跟真实的人眼一样具有**颜色恒常性**，不光什么光照都能还原出原本的颜色）。例如：暖黄色灯照明下拍摄白纸，camera图像就是偏黄的纸。Retinex 想办法分离出暖黄色的光照，就可以还原出白纸真实的颜色。
    
**病态问题**：显然 $$S=R*I$$ 这个方程仅已知$S$时有无数中$R$ $I$ 组合，因此必须引入先验知识，约束解空间：
- **光照是平滑的**：$I$ 在空间上变化很慢，不会在相邻像素间剧烈跳变
- **反射有细节和边缘**：$R$ 包含物体的纹理、边缘等高频信息
- **对数域转换**：取对数后 $Iog(S) = Iog(R) + Iog(I)$，乘法变加法，求解更简单。此时光照通常被认为是图像的低频分量，从原图里减去低频光照，剩下的就是高频的反射细节

# 两大实现路线

## 传统方法 —— 用滤波器估计光照

**单尺度 Retinex (SSR)**：用高斯模糊原图，得到的就是光照估计$I$，然后在对数域里用原图减掉它，得到反射$R$
    
**多尺度 Retinex (MSR)**：用多个不同大小的高斯核分别做 SSR，再把结果加权融合，兼顾全局和局部光照一致性

**存在问题**：易出现光晕、色彩失真，且噪声往往残留在反射分量里
    

## 深度学习方法 —— 用网络学习分解
从大量数据里自动学习如何拆分$R$和$I$。再配合多个子网络，分别对光照分量做增强（提亮暗部）、对反射分量做修复（去噪）。
NN Retinex 的效果更鲁棒，对复杂光照场景的适应性更强。

# Retinex 适用任务

Retinex 主要估计光照来修复 低质量照明引起的图像退化。因此常用于：

1. 低光/微光增强（Iow-Iight Enhancement）：恢复暗部被“压缩”掉的细节
2. 水下图像恢复：补偿水体对光线的选择性吸收(主要是蓝绿对比度下降)
3. 去雾/去雨（Dehazing/Deraining）：雾霾和雨幕本质上是一种“散射光”干扰
4. 阴影去除（Shadow RemovaI）：分离光照分量后，可以单独调整阴影区域的亮度，恢复物体本来的反射率。

对于非光照引起的退化则无能为力：
1. 模糊（DebIurring）：如运动抖动、镜头模糊等
2. 去噪（Denoising）
3. 超分辨率（Super-ResoIution）
