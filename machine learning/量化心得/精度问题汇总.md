
# concat
concat 用于将多个张量组装成一个。但是在模型量化时，如果组合的几个张量数据分布范围跨度太大，量化就需要更大的精度才能满足要求。
如下图 yolov8 输出是将 bbox跟得分组合成了一个张量，两个输入张量的分布范围差异很大，此时无法用int8量化。但是去掉concat，直接分别输出两个，两边都用int8量化精度损失也是可接受的
