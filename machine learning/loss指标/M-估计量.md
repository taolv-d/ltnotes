**M-估计量（M-estimator）** 是“最大似然型估计量”（Maximum-likelihood-like estimator）的缩写，由统计学家 Peter Huber 在 1964 年正式提出。

任何通过最小化 ∑iρ(yi−y^i) 或 E[ρ(r)] 来估计参数的损失函数，都属于 **M-估计量** 的范畴。

其中 ρ(r) 是一个关于残差 r 的函数。

你提到的所有损失函数，都是 **ρ(r)ρ(r) 的不同选择**。

```
M-估计量（M-estimator）
│
├── 经典 Lp 范数
│   ├── L2（最小二乘）
│   ├── L1（最小绝对偏差）
│   └── L0（理论上限）
│
├── 稳健 M-估计
│   ├── Huber 损失
│   ├── Smooth L1（≈ Huber 特例）
│   ├── Tukey 双权损失
│   └── Cauchy 损失
│
└── 退火/近似 L0
    ├── 退火 L0（Noise2Noise）
    ├── Geman-McClure
    └── Welsch 损失
```

# L1 与 L2 loss

假设有一张图像，对于任意像素位置 $i$，$y_i$ 代表该位置的真实像素值（Ground Truth），$\hat{y}_i$ 代表模型预测的像素值，$N$ 代表像素总数。

| 损失名称        | 数学公式                                                 | 别称                                | 异常值（离群点）敏感性                          | 收敛                                                           | 梯度公式<br>（反向传播）                                                                              |
| ----------- | ---------------------------------------------------- | --------------------------------- | ------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------------------------------------- |
| **L1 Loss** | $$ L_1 = \frac{1}{N} \sum_{i=1}^{N}(y_i-\hat{y}_i)$$ | **MAE** <br>(Mean Absolute Error) | 不敏感<br>梯度绝对值始终为 $1$（或 $-1$），模型更新步长稳定 | 梯度在零点不连续。解趋向于**稀疏**，有利于保留图像中的**边缘**和**高频细节**，常用于超分辨率任务。      | $\frac{\partial L_1}{\partial \hat{y}_i} = \text{sign}(\hat{y}_i - y_i)$<br><br>sign 是取符号函数 |
| **L2 Loss** | $$L2=\frac{1}{N}\sum_{i=1}^N(y_i−\hat{y_i})^2$$      | **MSE** <br>(Mean Squared Error)  | 极其敏感<br>大误差会主导loss                   | 梯度平滑连续。解是唯一的且趋向于平均值。**致命缺点**是生成的图像往往**模糊**，因为它倾向于对所有可能的解求平均。 | $\frac{\partial L_2}{\partial \hat{y}_i} = 2(\hat{y}_i - y_i)$                              |

| 损失函数        | 对误差（残差）的处理方式                                             | 图像效果                                                          |
| ----------- | -------------------------------------------------------- | ------------------------------------------------------------- |
| **L1 Loss** | 极小化误差的**绝对值**。它允许少数像素存在**很大的误差**（边缘处），只要绝大多数像素误差为 **0**。 | 既然边缘处误差大一点代价可控，模型就不会为了迁就边缘的像素差而去改动平坦区域的像素。**视觉表现：边缘锐利、背景干净。** |
| **L2 Loss** | 极小化误差的**能量**（平方和）。它倾向于把大误差拆分成很多个小误差。                     | 把边缘的剧烈落差"摊平"到周围的平坦区域。**视觉表现：边缘模糊、振铃效应。**                      |

**L1 L2 loss 的分工**

| 场景                  | 推荐 Loss       | 原因                          |
| ------------------- | ------------- | --------------------------- |
| **图像分类、逻辑回归**       | 交叉熵           | 概率分布匹配                      |
| **目标检测框回归**         | **Smooth L1** | 结合 L2 的平滑收敛 和 L1 的抗离群       |
| **图像超分/去噪（追求像素精度）** | **L2 (MSE)**  | 峰值信噪比（PSNR）指标与 L2 直接挂钩；平滑收敛 |
| **图像超分/去噪（追求视觉锐利）** | L1 + 感知Loss   | 利用 L1 的稀疏性保留边缘              |
| **传统数值分析、凸优化求解**    | **L2**        | **唯一有解析解，计算最快**             |

**关键技巧**：ddc∣c−y∣=sign(c−y)dcd​∣c−y∣=sign(c−y)（除了 c=yc=y 处不可导）。

# L2 loss 有这么多问题，为啥还要用？

**L2 loss 是一个数学性质最"好"、最稳定、最容易优化的凸函数。**

## 1. 唯一具有"封闭式解析解"的损失函数

在经典的线性回归问题中，如果使用 L2 Loss（最小二乘法），我们可以直接通过一个数学公式算出最优解：  
$$\hat{β}=(X^TX)^{−1}X^Ty$$

- **L2 的价值**：有且仅有 L2 Loss 能这么做。
    
- **L1 的困境**：如果你用 L1 Loss，**没有公式能直接算出答案**，必须依靠**梯度下降**或**线性规划**迭代逼近。

这意味着在很多**非深度学习的传统统计模型、信号处理、控制系统**中，L2 是唯一能给出**确定性、唯一解**的选择。

## 2. 唯一处处"平滑可导"的损失函数

神经网络的训练依赖**反向传播**。L2 Loss 的导数是 $2x$，这是一个**连续且线性变化**的函数。

- **L2 梯度**：误差越小，梯度越小 $\rightarrow$ 模型步长自动变小 $\rightarrow$ **更容易精确收敛到极小值点**。
    
- **L1 梯度**：导数永远是 $\pm 1$。即使在极小值点附近，梯度依然很大 $\rightarrow$ 模型在最优解附近**反复横跳**，很难稳定下来。

**这就是著名的"L1 收敛不稳定"问题。** 在深度学习中，优化器（如 Adam、SGD）最喜欢的就是 L2 这种**平滑、有良好 Lipschitz 连续性的函数**。这也是为什么很多时候即便最后评价指标是 L1，训练初期或者辅助分支也会挂一个 L2 Loss。

## 3. 高斯噪声假设下的"最大似然估计" [[../../TODO|TODO]]

这是 L2 Loss 存在的**统计学根基**。

- **L2 Loss** 等价于假设数据中的噪声服从**正态分布（高斯分布）**。
    
- **L1 Loss** 等价于假设数据中的噪声服从**拉普拉斯分布**。

自然界和物理传感器（如相机 CMOS）产生的**随机噪声**，绝大多数都完美符合**正态分布**。因此，如果你想让模型从带有随机高斯噪声的图像中恢复出原始信号，**数学上最优的估计器就是最小化 L2 Loss**。

# Smooth L1 Loss

由于 L1 在 0 点处不可导（震荡）、L2 容易梯度爆炸（对离群点），在目标检测的边界框回归中常用 **Smooth L1 Loss**：

$$Smooth L1(x)=
\begin{cases}
0.5x^2, & |x|<0.5 \\
|x|-0.5, & \text{otherwise}
\end{cases}$$

它在误差较小时是 L2 的平滑特性（容易收敛到精确值），在误差较大时退化为 L1 的特性（避免梯度爆炸）。

这个 0.5 确实不是随便取的，它是为了让 **Smooth L1 Loss** 在分段点 $x=1$ 处**不仅函数值连续，而且一阶导数也连续**。(这个推导很简单，将分界点设为未知数就好)

这就保证了在反向传播时，梯度是平滑过渡的，不会发生突变。

|Loss 类型|梯度公式 (绝对值)|梯度特性|
|---|---|---|
|**L2 Loss**|$\|2x\|$|误差越大，梯度越大，易**梯度爆炸**|
|**L1 Loss**|$1$|梯度永远是常数，零点处**不可导**|
|**Smooth L1**|$\|x\| \text{ (if } \|x\|<1\text{)} \text{ or } 1$|**零点平滑 + 远处截断**|
