- **全称**：Learned Perceptual Image Patch Similarity（学习感知图像块相似度）
    
- **本质**：一个**可微分的损失函数**，常用于训练生成模型。
    
- **原理**：它固定一个预训练的VGG[[../backbone/VGG、AlexNet|VGG、AlexNet]]网络，提取图像不同层的特征图，然后计算生成图像与真实图像在这些特征图上的距离。它模拟了人脑从低阶纹理到高阶语义的分层处理。
    
- **应用**：训练超分辨率、风格迁移、图像修复等生成模型时，用LPIPS代替像素级L1/L2损失，可避免结果过于平滑，保留更多感知上真实的纹理。

## 如何反向传播

深度学习的自动求导机制（autograd）并不关心某个中间网络是否被冻结。它只关心计算图是否连通。

**梯度流向哪里**：从损失 `L` -> VGG 各层（参数冻结，不更新） -> 输入图像 `x` -> 生成器。

### 1. 前向传播：提取特征
假设你有一张生成的假图 `x` 和一张真实图 `y`。

-   `x` 和 `y` 分别送入**冻结的** VGG 网络。
-   从网络的中间层（比如 `conv1_2`, `conv2_2`, `conv3_3` 等）取出特征图 `F(x)` 和 `F(y)`。
-   计算它们之间的 L2 距离（逐像素、逐通道），然后求和/平均，得到标量损失值 `L`。

### 2. 反向传播：梯度如何流动？

这里的核心是：**VGG 网络的权重是 `requires_grad=False`（即不计算梯度）。**

-   **梯度起点**：`∂L / ∂L` = 1。
-   **经过 L2 距离计算层**：这里完全是可微分的（加、减、乘、平方、开方）。这一层的梯度 `∂L / ∂F(x)` 可以顺利算出来。
-   **进入 VGG 网络**：现在梯度 `∂L / ∂F(x)` 要反向流过 VGG 的各层（从 `relu5_4` 一直传到 `conv1_1`）。
    -   **关键**：PyTorch/TensorFlow 在反向传播时，只是**机械地执行**矩阵乘法、卷积、ReLU 求导等运算。
    -   虽然 VGG 的权重 `W` 是固定的，但梯度仍然会穿过这些层，计算出 `∂L / ∂x`（即输入图像的梯度）。
    -   引擎**不会**计算 `∂L / ∂W`，因为 `W` 被标记为不需要梯度。所以 VGG 的权重不会更新。


# 棋盘效应 (Checkerboard Artifacts)


## 产生原因

这个现象是感知损失的反向传播特性与网络结构相互作用的结果：

*   **根源在于特定层的步长（Stride）**：产生格子的核心原因是网络中某一层或某几层的**步长（Stride）大于1**（例如步长为2）。在反向传播计算梯度时，这些层会进行**转置卷积**操作，当卷积核尺寸（Kernel Size）无法被步长（Stride）整除时，梯度在图像上的更新就会呈现不均匀的、规律的网格状叠加，如同“棋盘”一样。
*   **池化层（Pooling）也会加剧效应**：AlexNet 和 VGG 使用了大量的**最大池化层**。在反向传播时，梯度只会通过池化窗口内“最大值”的那个点进行回传，这导致了梯度更新的空间不连续性，进一步固化和放大了网格感。

### VGG 的格子为什么比 AlexNet 大？

这主要由两个因素决定：

1.  **网络深度与感受野（Receptive Field）**：VGG 比 AlexNet 深得多（如 VGG19 有19层），其高层神经元的**感受野（即对应回原图的一块区域）** 远比 AlexNet 的大。大的感受野意味着对底层的梯度调制模式更宏观，因此反映在图像上的网格尺寸也就更大。
2.  **卷积核尺寸的差异**：AlexNet 使用了较大的卷积核（如 11x11、5x5），这种大尺寸的卷积核对网格有一定的平滑作用。而 VGG 统一使用极小的 **3x3 卷积核**，这种规整的小卷积核更容易在反向传播中形成清晰、规律、高密度的网格纹理。

### 解决方案与缓解措施

*   **调整损失计算层（Loss Layer）**：这是最有效的方法之一。不要使用网络最深层的特征（如 `conv5`），尝试改用较浅层的特征（如 `conv2` 或 `conv3`）来计算感知损失。浅层特征的感受野更小，语义信息更局部，能显著减少宏观的规则网格。
*   **使用替代性预训练网络**：换用更现代的网络作为损失网络，比如 **SqueezeNet** 或 **ResNet** 的早期层。特别是 ResNet，由于它没有使用步长大于1的卷积来下采样（改用步长为1的卷积+池化，或空洞卷积），其内部的网格效应通常比 VGG 要轻得多。
*   **在损失网络前加模糊（Blur）**：这是实现起来最简单的技巧。在将生成图像和目标图像送入 VGG/AlexNet 计算损失之前，先用一个极小的（如 3x3 或 5x5）高斯模糊核对图像进行轻微平滑。这能打乱像素级的规整对齐，显著破坏网格形成的条件。
*   **混合损失函数**：不要单纯依赖感知损失。增加 **L1 损失**（平均绝对误差）或 **SSIM 损失**（结构相似性指标）的权重。L1 损失是对像素级的直接约束，它天然反对任何形式的规律性噪声（如棋盘格），可以有效中和感知损失带来的副作用。
