论文：_Rapid Object Detection using a Boosted Cascade of Simple Features_
本文是微软跟三菱合作提出的目标检测算法。这篇论文对计算机视觉的影响十分巨大，是计算机视觉领域的奠基之作。在当时算力受限的设备上，难以训练CNN网络，同时也缺少数据集。

本文算法目的是提出一个通用目标检测算法，文章以人脸检测为主。当前很多3A 算法检测人脸也会使用本文方案。

本文很像一个人工设计的模型（人工特征工程+集成学习）：
- 手动设计Haar 特征——现代CNN通过学习提取特征
- AdaBoost 特征选择——现代网络的剪枝、神经架构搜索
- 最后是NMS，剔除重复检测框

# Haar 特征

本文的 Haar 特征如下图所示，它们可以检测**边缘、线性、中心、对角线**特征
![[attachments/Pasted image 20260720224207.png|357]]
对于一个24* 24的检测窗口，这些特征通过**缩放、平移**生成超16万个特征。
为了处理这么多特征，算法引入**积分图**加速计算。但是还是必须要删减特征才行。

# AdaBoost

AdaBoost 算法的目的就是精简这16万个特征。具体做法如下：
1. 它会将这16万个特征逐一应用到成千上万张标注好的训练图片上。
2. 对每个特征，算法会找出一个最佳的**阈值**，用来区分人脸和非人脸，并计算这个特征带来的**分类错误率**。
3. 错误率最低的特征，就被认为是**最有效**的特征。
4. 通过多轮迭代，AdaBoost会选出那些分类能力最强、且彼此互补的特征组合。
5. 最终，它筛选出的**前200个**最有效的特征，组合起来就能达到**95%** 的检测准确率。

# 算法局限

本算法在当时算力局限的情况下确实经验，但他也有很多局限性：
1. 泛化能力差
2. 缺乏旋转不变性
3. 表达能力有限（难以识别复杂纹理）

**旋转不变性改进**
旋转不变性的改进其实有很多套路：
1. 如果已知旋转角度不大，可以直接加数据训练
2. 多个集联分类器，每个处理不同角度
3. 直接旋转图像（会增加计算量）
4. 设计旋转不变的特征
