原文：A Review of Recent Advances in High-Dynamic-Range CMOS Image Sensors
本文回顾了 2019~2025的HDR技术

开始前，先明**确动态范围受限的根本原因**：
- **饱和容量（FWC）** 有限
- **噪声**（特别是暗噪声）限制了最小可测光强

# 提升DR的三大技术路线

论文将HDR技术分为三类：
- **增加像素满阱容量（FWC）**，容纳更多光电子，避免溢出，具体又细分为三类：
	- **Multiple Conversion Gain（多转换增益）**： 通过改变浮置扩散节点的电容（可选大电容或小电容），实现高增益（小电容，弱光）和低增益（大电容，强光）切换。
	
	- **LOFIC（横向溢出积分电容）**：在PD旁边集成大电容，吸收溢出的电荷(溢出通道+存储电容)。存在问题：切换点 SNR 下降，KTC 噪声增加
	
	- **其他结构**：FeFET（铁电FET）3D 堆叠

- **多曝光时间**，通过不同曝光时间/编码的图像合成HDR。
	- **Coded Exposure**：将一帧分成多个短子曝光，每个像素根据光强调节曝光编码。
	-  **Split-PD**：一个像素内分成大PD（弱光）和小PD（强光）。
	- **Time-to-Saturation (TTS)**： 记录每个像素达到饱和的时间，用于反推光强。

- **优化信号输出响应**
	- **SPAD（单光子雪崩二极管）**：光子计数模式，无噪声读出的HDR。
	- **Linear-Logarithmic（线性-对数）像素**：弱光时线性响应（高SNR），强光时对数响应（避免饱和）。

**这些细分技术的对比**，目前看到的已经有商用的方案：DCG、LOFIC、多曝光（跟coded Exposure 有差异）、Split-PD(4PD)

| 技术类型           | DR能力 | SNR    | 分辨率影响 | 成本  | 主要弱点        |
| -------------- | ---- | ------ | ----- | --- | ----------- |
| Multiple-CG    | 中    | 中      | 无     | 低   | PD容量仍有限     |
| LOFIC          | 高    | 低（切换点） | 无     | 低   | KTC噪声、SNR下降 |
| Coded Exposure | 高    | 高      | 无     | 中   | 依赖算法        |
| Split-PD       | 中    | 中      | 下降    | 中   | 像素分割降低敏感度   |
| TTS            | 高    | 高      | 无     | 中   | 弱光信号难读      |
| SPAD           | 极高   | 极高     | 下降    | 高   | 功耗、面积、带宽    |
| Linear-Log     | 高    | 中      | 无     | 中   | 非线性、开关点不确定  |

## 增加满阱容量

### DCG 相关技术

![[attachments/Pasted image 20260506151234.png]]



**(a) 四共享像素 + DCG**
- **传统 Tetra cell**：4 个 PD 共享一个 FD，弱光下合并信号提升 SNR。
- **加入 DCG**：通过额外开关 + 电容，改变 FD 总电容。
- **HCG（小电容）** → 高灵敏度，适合弱光。
- **LCG（大电容）** → 低灵敏度，适合强光，防止饱和。
- **本质**：**通过开关控制 FD 电容，实现单帧内双增益读出**。

 **(b) PPD + 浮栅 + DCG**
- **两次读出同一电荷**：
    - 先通过**浮栅**（大电容）读出 → LCG（强光信号）
    - 再通过**PN 结 FD**（小电容）读出 → HCG（弱光信号）
- **不破坏电荷**（非破坏读取），保证两次读出的物理量一致。
- **优势**：单次曝光、无运动伪影。


 **(c) 传统 DCG 像素结构**
- **标准 DCG 结构**：FD 节点 + 一个可选附加电容。
- **问题**：切换瞬间会引入**电荷注入**（时钟馈通）和 **KTC 噪声**。
- **改进**：**HCG 和 LCG 信号路径分离处理**，分别降噪后再融合。

**带有 DRAM 的 DCG 像素**
- **每像素一个 DRAM 单元**：存储 1 bit 模式选择（HCG / LCG）。
- **工作流程**：
    1. 预读出（快速扫描）
    2. 判断光强（写入1bit 模式开关）
    3. 自适应选择 CG 模式（根据1bit模式开关）
    4. 正式读出
- **优势**：
    - 自适应，不用全局统一模式（每个pixel一个配置），但这会给ISP处理带来新问题吧？
    - 帧率翻倍
    - 降低功耗（通过列级触发信号控制）

 **(e) FDTI + P 型保形掺杂**（半导体层面改进，不深入研究）

> 这是 **FWC 增强技术**，不属于严格意义的 DCG，但常与 DCG 配合使用。
- **FDTI**：防止相邻像素串扰。
- **P 型保形掺杂**：优化 PD 电势分布，提高 FWC。
- **配合 DCG**：进一步提升 DR。

**(f) FD 共享技术**
- **四个 PD 共享一个 FD**：节省面积，提高 FWC 利用率。
- **四分之一环形 SF**：提高驱动能力，降低噪声。
- **目的**：在小像素（1.28μm）中保持 DR。

### LOFIC

![[attachments/Pasted image 20260506154320.png]]

**(a): 带有 LOFITreC 电容阵列的像素**
- **LOFITreC**：利用**深沟槽技术**在硅片里做出的**大型三维电容器**，面积利用率极高。
- **工作流程**：PD收集光电子 → PD满后，多余的电子通过“溢出路径”流向 LOFITreC 这个大电容 → 读出时，同时读出 PD 和 LOFITreC 中的电荷。
- **优点**：在巨大的FWC下仍保持了可观的填充因子（光感面积占比）。


**(b): 带有两级 LOFIC 的像素**
- **两级 LOFIC (Two-Stage LOFIC)**：图中可以看到PD后面串联了两个电容。如果一级不够，可以再装一级，把溢出电荷进一步转移，实现**超大的等效FWC**。
- **高压存储记忆 (Voltage Memory)**：用于全局快门，先把信号存储起来再统一读出。
- **应用场景**：高速X射线成像，既需要高帧率，又需要大动态范围来分辨不同厚度的组织。


**(c): 带有中转换增益（Medium-CG）的LOFIC像素**
- **Medium-CG**：之前DCG和LOFIC只是在“大电容”（LCG）和“小电容”（HCG）间切换。这里增加了一个“中电容”状态。
- **优点**：避免从极低增益一下子跳到极高增益，使**SNR随光强变化更平滑**，减少人眼察觉的噪点突变。
- **应用场景**：汽车自动驾驶，需要应对隧道入口这种光线剧烈变化的场景。


**2(d): 双采样电路**
- **问题**：LOFIC会读出两个信号（一个来自感光度高的PD，信号微弱；一个来自大电容，信号巨大）。两者大小差上千倍，无法用一个放大器/ADC正常处理。
- **解决方案**：设计两条信号通路：一条用于微弱信号（高增益），一条用于强信号（低增益）。然后图中用一个巧妙的**开关矩阵**把这“看似两套”的电路合二为一，节省面积。

## 多曝光合成

### Coded Exposure

个人认为方案d更可行（仅从原理上看）

| 子图      | 核心创新点           | 解决的核心问题         |
| ------- | --------------- | --------------- |
| **(a)** | **双抽头**         | 避免子曝光期间的电荷损失    |
| **(b)** | **自动曝光+黑电平校正**  | 自动确定最佳曝光参数      |
| **(c)** | **乒-乓模拟存储器**    | 曝光与读出并行，提高帧率    |
| **(d)** | **自复位+行交织**     | 小像素实现极大等效FWC    |
| **(e)** | **分块独立曝光+3D堆叠** | 灵活控制，高帧率HDR     |
| **(f)** | **RFDC（时间戳转换）** | 曝光同时完成ADC，无额外时间 |


![[attachments/Pasted image 20260506155123.png]]


**(a): 双抽头编码曝光像素（Dual-Tap Coded-Exposure）**

- **单抽头 vs 双抽头**：单抽头在子曝光关闭时会丢弃电荷，造成信号损失。双抽头有两个收集节点（抽头A和B）。每个子曝光期间，电荷可以选择流向A或B，**不会丢失**。
- **工作流程**：帧分成900个子曝光 → 每个像素根据当前光强，决定这口“饭”盛给A还是B → 曝光结束后，A和B分别累积了不同亮度的信息 → 融合成HDR图像。
- **优点**：速度快（39,000子曝光/秒），能捕捉快速运动，无运动伪影。


**(b): 自动曝光（AE）和黑电平校正（BLC）
- **核心问题**：如何确定每个像素的最佳曝光时间？
- **解决方案**：用不同的曝光时间（8种）快速预采样，判断光强范围，然后自动选择最合适的那一档。
- **黑电平校正**：确定“纯黑”对应的参考电平，保证暗部细节不丢失。
- **优点**：避免了用户手动设置曝光，适合光照动态变化的场景。


**(c): 带有嵌入式模拟存储器的像素**
- **传统问题**：**全局快门需要先曝光完所有像素，再一列列读出**。曝光和读出**串行**，限制了帧率。
- **乒-乓缓冲**：每个像素有两个存储节点（乒乓）。曝光k+1帧的同时，读出k帧。
- **优点**：**帧率翻倍**，同时保持全局快门（无运动畸变）。


**(d): 行交织读出的时序图**
- **自复位（Self-Reset）**：每个像素在曝光期间可以**多次复位**。每次快满了就复位，并记录一次“溢出事件”。
- **信号重建**：信号量 = 复位次数 × FWC + 最后一次未满的残余电荷。
- **行交织**：不同行的复位和读出错开时间，避免冲突。
- **优点**：让小像素也能拥有巨大的等效FWC。


**(e): 堆叠式CIS架构（有点像QPC的HDR模型）**
- **分块控制**：把整个像素阵列切成许多小块，每块可以有自己的曝光时间。
- **适应性强**：画面中亮区域用短曝光，暗区域用长曝光。
- **3D堆叠**：把像素层和逻辑控制层堆叠起来，实现高速、低延迟的控制。
- **优点**：非常灵活，可以在高帧率和HDR之间权衡。


**(f): 回归式通量-数字转换**
- **传统ADC**：需要时间把电压转换成数字码。
- **RFDC**：用一个正弦波作为参考，当像素电压和正弦波相交时，记录**交点的时刻**。通过时刻反推光强。
- **优点**：**曝光的同时就在转换**，不需要单独的读出时间，速度极快，适合捕捉快速变化的光。

### Split-PD

| 子图      | 核心创新点             | 解决的问题                 | 典型应用      |
| ------- | ----------------- | --------------------- | --------- |
| **(a)** | SPD + LPD + LOFIC | 极宽DR（140dB）           | 手机、汽车     |
| **(b)** | FDTI + 表面修复       | 高温下的SNR恶化             | 汽车（105°C） |
| **(c)** | “H-I”大小像素组合       | HDR + 颜色 + LED检测      | 车规相机      |
| **(d)** | 2×8共享 + 垂直TG      | 高分辨率下的FWC             | 64M像素手机   |
| **(e)** | Quad-PD + 溢出共享    | 极小像素（0.5μm）HDR + PDAF | 超高分辨率手机   |
| **(f)** | Twisted-PD        | 全方向自动对焦               | 通用AF      |


![[attachments/Pasted image 20260506155659.png|581]]
![[attachments/Pasted image 20260506155719.png|527]]


**(a): Split-PD + LOFIC 像素**
- **两种PD**：
	- **SPD（小PD）**：灵敏度低，用于强光场景。
	- **LPD（大PD）**：灵敏度高，用于弱光场景。
- **四种工作模式**：通过组合SPD/LPD和HCG/LCG，实现四档灵敏度切换。
- **LOFIC电容**：专门用于SPD，让它在强光下也不会饱和（等效FWC达25Me⁻）。
- **优点**：覆盖极宽的亮度范围（140dB）。

**(b): 带有FDTI的子像素**
- **FDTI（全深度深沟槽隔离）**：在PD之间挖深槽，防止光生电子串扰到相邻像素。
- **高温性能**：汽车等应用需要在高温下工作（105°C），高温会加剧漏电流。表面缺陷修复工艺可以钝化硅表面，减少漏电。
- **优点**：解决了Split-PD在高温下的SNR恶化问题。

**(c): “H-I”共享像素结构**
- **“H”结构（小像素）**：1.5μm，灵敏度低，用于强光/颜色。
- **“I”结构（大像素）**：3.0μm，灵敏度高，用于弱光。
- **组合目的**：同时实现HDR和彩色成像。
- **优点**：适合需要同时识别LED闪烁（如车灯）和弱光环境的车规应用。
- - **Clear像素**：**没有滤光片**，相当于“全透”窗口，能接收所有波长的光（红+绿+蓝+红外）。它的**灵敏度最高**，但**没有颜色信息**。
- **Gray像素**：有一层**中性密度滤光片**，相当于给像素戴上了一副“**墨镜**”。它能均匀地降低所有波长的光强（比如衰减到10%），但**不改变光的颜色**。它的**灵敏度被刻意降低了**，但仍然保留了部分光谱信息，通常用于辅助颜色重建或进一步提升动态范围。

**(d): 八共享像素结构（可切换CG）**
- **垂直传输门**：把TG从横向变成纵向，节省面积，同时可以做更深的PD来存储更多电荷。
- **2×4共享**：8个像素共享一套读出电路，每个像素的实际FWC是物理FWC × 共享数量（因为分时读出）。
- **可切换CG**：通过三个复位管串联实现三档CG切换。
- **优点**：在高分辨率（0.7μm像素）下维持可用的DR。

**(e): Quad-PD 像素**
- **Quad-PD**：一个2×2的PD阵列组成一个“逻辑像素”，用于相位检测自动对焦（PDAF）。
- **溢出路径共享**：当一个PD满了，电子可以流向相邻的PD，提高等效FWC。
- **8像素共享FD**：进一步节省面积。

**(f): Twisted-PD（扭曲PD）俯视图**
- **传统双像素（Dual-PD）**：每个像素分成左右两半，只能检测水平方向的相位差。
- **扭曲PD**：PD的形状和分割方式经过优化，可以检测**水平和垂直**两个方向的相位差，实现全方向对焦。
- **与HDR的关系**：这种结构也属于Split-PD的一种变体，可以结合HDR技术使用。

### Time-to-Saturation (TTS)

以前直接测量电压，现在改成与已知量比较+记录时间，推测信号强度。改为间接测量。
但是这种曲线救国的方案电路很复杂（计数器、ADC、频率发生器等）

| 子图      | 核心创新点               | 解决的问题      | DR示例     |
| ------- | ------------------- | ---------- | -------- |
| **(a)** | 斜坡电压 + 比较器          | 曝光同时完成时域转换 | -        |
| **(b)** | **TTS + 线性ADC双量化**  | 弱光下TTS失效问题 | 107dB    |
| **(c)** | 令牌读出 + 重复扫描         | 降低功耗       | 96dB     |
| **(d)** | **PFM-ADC（脉冲频率调制）** | 数字输出，抗噪声   | 96-120dB |

![[attachments/Pasted image 20260506161455.png]]


**(a): TTS 工作原理示意图**
- **工作流程**：
	1. 曝光开始，PD电压随着光强线性下降（或上升，取决于电路结构）。
	2. 同时，一个全局斜坡电压也在变化。
	3. 当PD电压**跨过**斜坡电压时，比较器翻转，**记录下此刻的时间**。
	4. 时间越短 → 光越强；时间越长 → 光越弱。
- **关键组件**：每个像素需要一个**比较器**（比较PD电压和斜坡电压）和一个**计时器**（记录翻转时刻）。
- **优点**：曝光和“模数转换”同时进行，无需额外ADC时间。

**(b): TTS 量化操作时序图**
- **TTS的盲区**：TTS在**弱光**下有一个严重问题——如果光太弱，PD可能在整帧曝光时间内都**从未饱和**，那么就没有“饱和时间”可记录。
- **解决方案：双量化**
	- **强光（PD会饱和）**：用TTS，通过记录饱和时间来确定光强。
	- **弱光（PD不会饱和）**：用传统线性ADC，通过读取最终电压来确定光强。
- **图中的波形**：上半部分展示PD电压随时间下降，下半部分展示比较器翻转时刻。

**(c): 带两级动态比较器和令牌锁存的像素**
- **令牌读出（Token Readout）**：一种逐像素扫描的读出方式，每次只有一个像素激活比较器，其他像素等待。
- **重复扫描**：多次扫描整个阵列，每次使用不同的阈值电压。
- **为什么省电**：时间模式（Time-Mode）电路主要使用数字逻辑（定时器、比较器），避免了功耗较高的线性ADC。
- **代价**：读出速度较慢（因为要多次扫描）。
- **优点**：DR翻倍（48dB → 96dB），功耗极低。

**(d): 带TTS的数字像素CIS**
- **脉冲调制频率ADC（PFM-ADC）**：PD电压变化控制一个振荡器的频率，光越强振荡越快。通过**计数振荡次数**来测光强。
- **本质上是TTS的一种变体**：不是测“一次饱和的时间”，而是测“单位时间内的饱和次数”。
- **数字像素**：每个像素内部集成了比较器、振荡器、计数器（至少部分）。
- **3D堆叠**：像素阵列和逻辑电路分层制造，实现小像素尺寸。
- **优点**：数字输出，抗噪声强；通过增加计数位宽可扩展DR（96dB → 120dB）。

## 优化信号输出响应

### SPAD（单光子雪崩二极管）

SPAD本身的DR能力很强，但实现HDR需要**大计数器和高速电路，导致面积和功耗失控**。这些技术，本质上都是在用各种方法来**“压缩”计数器的位数**，节省面积跟功耗。

| 子图      | 核心创新点           | 解决的问题   | DR示例  |
| ------- | --------------- | ------- | ----- |
| **(a)** | 像素级曝光控制 + 自适应充电 | 功耗控制    | 143dB |
| **(b)** | 子帧采样 + 外推计数     | 减少计数器位宽 | 124dB |
| **(c)** | 时间编码外推（溢出时间）    | 降低硬件复杂度 | 113dB |
| **(d)** | MSB片外存储         | 缩小像素间距  | 120dB |
| **(e)** | 片上多帧求和          | 减少像素内电路 | 126dB |

**(a): 3D BSI SPAD + 像素级曝光控制**
- **像素级曝光控制**：每个像素独立判断当前光强，如果计数过快（光太强），就**主动缩短曝光时间**，避免计数器溢出。
- **自适应时钟充电**：SPAD需要周期性充电来恢复探测能力。弱光下降低充电频率（省电），强光下提高充电频率（跟上光子速率）。
- **3D堆叠**：SPAD器件层和电路层分开制造，再键合在一起。这样SPAD可以做在最优的衬底上（提高探测效率），电路可以用先进工艺（缩小面积）。
- **电荷聚焦**：优化SPAD结构，让光生载流子更容易触发雪崩。

**(b): 子帧采样 + 外推光子计数**
- **问题**：强光下需要很大的计数器（如30位），占用面积和功耗。
- **解决方案**：
	1. 把一帧分成很多个**短子帧**（如256个）。
	2. 每个子帧只用一个**小计数器**（如9位）计数。
	3. 如果某个子帧计数器**溢出了**（说明光极强），就**外推估算**：实际光子数 ≈ 溢出次数 × 子帧饱和值。
	4. 最终HDR值 = 所有子帧真实计数 + 外推计数。
- **为什么有效**：不需要做30位的大计数器，只需要9位小计数器 + 累加逻辑。
- **代价**：需要存储每个子帧的结果，增加存储需求。
- **核心贡献**：**用小计数器实现大DR**，大幅降低像素面积和功耗。

**(c): SPAD X射线探测器 + 时间编码外推计数器**
- **X射线成像的特点**：动态范围极大（从骨骼到软组织），剂量率变化剧烈。
- **时间编码外推**：与(b)类似，不是记录“溢出了几次”，而是记录**第一次溢出发生的时间**。（如果在曝光1/10的时间就溢出了，那么估算总光子 ≈ 10 × FWC）
- **无缝全局快门**：所有像素同时曝光和读出，没有运动伪影。
- **核心贡献**：**用时间编码替代计数**，进一步降低硬件复杂度。

**(d): 聚类多周期时钟充电 + MSB片外存储**
- **PFE（Pixel Front-End，像素前端）**：包括SPAD、淬火电路、一个小计数器。
- **问题**：做大计数器需要大面积的数字电路，放在PFE内会增大像素尺寸。
- **解决方案**：
	- 每个PFE只放一个**小计数器**（低位，如8-10位）。
	- 当小计数器**溢出**时，溢出的次数（高位）不存储在PFE内，而是通过阵列扫描送到**外部的SRAM**中。

**(e): 片上帧求和**
- **问题**：单帧内想达到高DR，需要大计数器，占用面积。
- **解决方案**：
	- 每帧只用**小计数器**（如10位）记录光子数。
	- 连续拍摄**多帧**，把N帧的结果**累加**起来。
	- 最终DR = 单帧DR + 10log10(N)。
- **时间换空间**：用帧率换取DR，适合静态场景。

**(a): 线性与对数操作结合的概念图**
在光强超过某个阈值后，输出信号从线性增长平滑过渡到对数增长，从而无限延缓了饱和点。

**(b): FDSOI（全耗尽绝缘体上硅） 器件的能带图**
- **原理很简单**：这种硅晶体管下面有一层“绝缘层”（BOX），再下面还有一个“背栅”。
- **在暗光/弱光下**（图中上半部分）：晶体管被设计工作在**线性模式**，像一个标准的光电管，灵敏度高，信号跟光强成正比。
- **在强光下**（图中下半部分）：大量的光生电荷会把背栅电压“顶”上去，**动态地改变晶体管的阈值电压**，使其自动切换到**对数模式**。

**(c): PD-MOS (光电二极管体偏置MOSFET) 示意图**
- **结构**：把一个普通的光电二极管（PD）和MOS晶体管的“身体”（体）接在一起。
- **原理**：
    - 弱光时，PD收集的电荷很少，对晶体管的“体”电位影响不大，它就是一个**线性放大器**。
    - 强光时，PD收集的电荷很多，会直接抬高/压低晶体管的“体”电位，从而**改变其阈值电压**，使晶体管进入**对数模式**。
- **优点**：与标准CMOS工艺兼容，不需要FDSOI这种特殊衬底。
- (b) 和 (c) 的底层逻辑是一样的：通过光强来改变晶体管“身体”的电平，让它的工作模式从“线性”变成“对数”

**(d): PD-MOS的明暗差分结构**
- 对(c) PD-MOS的改进。通过差分结构（一个测光，一个测暗参考），可以消除共模噪声（如温度漂移、工艺偏差），提高输出稳定性。
- 两个结构相同的PD-MOS，一个正常感光，一个被遮挡作为暗参考。两者输出相减，抵消噪声。

**(e): 双列线性PD-MOS**

**(f): DCG + 对数响应结合**
- 弱光：DCG的HCG模式 + 线性响应，保证SNR。
- 中光：DCG的LCG模式 + 线性响应。
- 强光：切换到对数响应，防止饱和。
- 三种模式平滑切换，覆盖极宽的光强范围。
