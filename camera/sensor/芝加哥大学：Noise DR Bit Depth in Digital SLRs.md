# 1. 该文章主要讨论以下几种噪声：

1. 散粒噪声（P）: 服从泊松分bu

2. 读出噪声（R）：全黑时测量

3. Pattern noise: 占噪声比例小，但视觉影响大

4. 热噪声：由热电子引起，随曝光时间增加而增加，但曝光足够大时部分pixel饱和因而噪声特性曲线的斜率下降。另一个热源是放大器

5. PRNU(W):像素相应不均匀，即使无散粒噪声、热噪声也会因PRNU引入噪声。PRNU随曝光正比例增加，长曝光影响大。可通过乘系数矫正。

   1. 亮处PRNU能达到散粒噪声+热噪声

   2. 低曝光时主要是散粒噪声

   3. 极暗时主要是读出噪声

6. 量化噪声：模拟转数字时的四舍五入，在噪声中的占比很小。当噪声超过量化步长时可以忽略。



# 2. 如何测量散粒噪声、读出噪声、PRNU

1. 差分法：

   1. 拍两张均匀离焦图

   2. 相减 剪掉PRNU、图像梯度，同时噪声增加

   3. 公式待补充

   4. &#x20;在不同亮度下拟合 y=ax+b

      1. 其中 b = 1/增益（增益：g 电子数/DN）

      2. a读出噪声

2. PRNU计算：

   法一：由N^2=P^2+R^2+W^2，可得

   法二：拍50张均匀离焦图叠加：PRNU+OB

   &#x20;  拍50张暗场图叠加：只有OB

   两者相减即可

3. DSNU: 暗场图，所有pixel的标准差

4. 读出噪声：两暗场图相减的差值图的标准差/根号2

5. FPN: DSNU+PRNU



# 3. 信噪比、动态范围与ISO

1. PRNU随亮度增加而增加，w=k\*s (s为信号强度)，因此会限制信噪比，信噪比上限为 s/(ks) = 1/k

2. 工程定义：最大信号（实测）/ 读出噪声。不实用，人眼无法分辨

3. 对于单放大器：R^2=(GR0)^2 + R1^2

   1. R0 ISO 放大器上游引入

   2. R1 ISO 放大器下游引入

4. 对于双ISO放大器：R^2=G2^2 \* ((G1R0)^2 + R1^2)  + R2^2

   1. 低ISO时噪声主要来自下游，R1贡献大，R0小，R2贡献小

   2. 低ISO时放大器是主要噪声

   3. R0在高ISO时影响大

5. 高ISO时使用硬件放大（模拟）还是软件（数字）放大对最终噪声无影响

   1. 低ISO时：优先硬件放大，在AD前提升信息，压倒读出噪声，减弱量化的影响

   2. 高ISO时：读出噪声已经很大，即使硬件放大噪声也非常大，**噪声水平远超量化步长**



# 4. 噪声的影响

1. 图像出现色带变化（平滑 -> 阶梯）

   1. 噪声抖动机制失灵，正常是噪声>量化误差，抹平了量化引入的边界，实现平滑过渡

   2. 量化步长比噪声水平精细很多是浪费

   3. 量化步长比噪声水平高会引入色带

   4. 理想情况是噪声略大于量化步长

   5. 后期处理（gamma, 降噪+重采样，低bit显示）也可能引入色带。中值滤波会降低细节，也会在均匀过度的图像上引入色带，中值滤波在更高bit上表现更好

   6. 高bit量化有时无必要

2. S/N 与正确曝光

   1. 优先调曝光、光圈以获得更大曝光，随后才是ISO。相比不提ISO，提高ISO能够抑制噪声，原因是放大器下游引入的噪声被抑制。

   2. 像素合并提高snr,四合一snr\*2

3. 有损压缩：高亮压缩 暗部保持

   1. 高亮区散粒噪声远远大于量化步长->低 bit ADC

   2. 低光区散粒噪声小，需要更精细的量化步长->高bit ADC

   3. 采用线性接二次曲线的压缩方式

4. 降噪方法：

   1. CDS双采样，无损降噪

   2. FPNC: 减暗场（与实拍完全相同的拍摄条件）

   3. PRNU: 平场增益图
