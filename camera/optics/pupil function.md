## pupil function
pupil function 中文常翻译为 光瞳函数，孔径函数。它的数学表达式为：
$$
P(x,y)=A(x,y)\cdot e^{i\cdot \phi(x,y)}
$$
- $A(x,y)$：**振幅透过率函数**（0到1之间）。描述入瞳平面上不同位置允许通过多少光。
- $e^{i\cdot \phi(x,y)}$：**相位函数**。描述光线经过入瞳不同位置时产生的相位延迟（像差）。
- $(x,y)$：入瞳平面上的坐标。

孔径函数可以PSF([[PSF]])
$$
PSF(u,v)=|\mathcal{F\{\mathnormal{P}(x,y)\}}|^2
$$
孔径函数直接决定了衍射极限下的成像质量：
- 圆形孔径 → 艾里斑（sinc² 型）。
- 六边形孔径 → 六边形衍射斑，会有6个星芒

## 星芒

星芒的本质是**衍射**。当光线通过一个**有尖锐棱角**的开口（光圈叶片）时，光波在棱角处发生衍射，产生干涉，形成沿着棱角方向向外延伸的明亮条纹。

星芒的形状和数量，**直接由孔径函数（Pupil Function）中的振幅分布（即入瞳的形状）决定**。

对于**正多边形**光圈的星芒数量，规律如下：

| 光圈形状    | 叶片数 NN | 星芒数量 | 例子                  |
| ------- | ------ | ---- | ------------------- |
| **偶数边** | 偶数     | N条   | 6片 → 6星芒；8片 → 8星芒   |
| **奇数边** | 奇数     | 2N 条 | 5片 → 10星芒；9片 → 18星芒 |

**原因**：偶数边形的对边平行，衍射条纹在对边方向会重叠，星芒数量等于边数。奇数边形的边没有平行对边，衍射条纹在两侧都出现，所以数量翻倍。
