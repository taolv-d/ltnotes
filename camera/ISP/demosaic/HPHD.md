##  1. 插值G 

**构造方向图（选水平还是竖直还是两个混合）**
差值G 之前，先构造一个方向图（hpmap），方向图的作用就是指出当前点到底是用水平的估计值、还是垂直的估计值，还是两个都用，判断依据也是谁的结构更强（说明有纹理），谁就不适合差值。

先在垂直方向上计算一个宽支撑高通响应（提取高频，过滤掉缓慢变化的低频）：

```cpp
temp = abs(
    (x[-5] - x[+5])
  - 8  * (x[-4] - x[+4])
  + 27 * (x[-3] - x[+3])
  - 48 * (x[-2] - x[+2])
  + 42 * (x[-1] - x[+1])
)
```

然后在 9 点窗口里求：

```text
avg = 局部平均高通响应
dev = 局部高通波动
```

再得到垂直方向的平滑高通图：

```text
hpmap_vertical = avgL + (avgR - avgL) * devL / (devL + devR)
```

同样可以得到 v 方向的，两个方向比较：那个方向数值大，说明有纹理、边缘，不适合差值，那就应该用另一个方向。如果两者差不多，那就两者混合

**G插值**

色差估计比较简单：
```cpp
// 水平
g2 = G[right1] - 0.5 * C[right2]
g4 = G[left1]  - 0.5 * C[left2]

// 竖直
g1 = G[up1]   - 0.5 * C[up2]
g3 = G[down1] - 0.5 * C[down2]
```

再算两个方向的权重：

```
e = 1 / (局部差分和)
```

插值：

```cpp
// 水平 或竖直
G = 0.5 * C[x] + (e2 * g2 + e4 * g4) / (e2 + e4)

// 混合
G = 0.5 * C[x] + (e2 * g1 + e2 * g2 + e3 * g3 + e4 * g4) / (e1 + e2 + e3 + e4)
```
