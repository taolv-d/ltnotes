这本书中提到的绝大部分工程实践方案已经被取代，但本书聚焦的最底层的原理仍十分重要（第一性原理）

这是一本2009年的书，放在当下看已经有一些过时了，但书中对CFA的一些分析符合第一性原理，放在当下仍值得一读，完整的推导设计信号处理的知识，这里不会深入展开（或者后续补充）。此外很多算法模块的设计虽然没有沿用至今，或被神经网络算法秒杀，但其思想在ISP中仍有体现，因此也值得记录。这里不会纠结细节。

# CFA 的分析与去马赛克算法

分析方向包括：空域、频域、时域、光谱四个方向

1. 为什么要用CFA 阵列？

便宜，使用CFA阵列前都是三个传感器

* CFA 设计的原则

  1. 色彩空间

  2. 排列方式（理论上随机排列能够避免顺序排列引入的artifices 但算法复杂）

* 当前常见的CFA 存在的问题

  1. 首先作者通过频域分析，发现CFA 是将亮度信号作为低频，将色差信号调制到了水平和垂直方向。但水平垂直方向是自然图像中能量最集中的部分，造成亮度信号与色差信号混叠，因此无法很好的还原出真实信号。

  2. 这也是为什么去马赛克算法都会引入非线性操作来加强边缘

  3. 作者设计了一个特殊的CFA阵列，让色差信号均匀分布在高频，且避开水平垂直方向（即不使用RGB滤光片，而是其他颜色，每一个通道可以看成是RGB 三个分量不同权重的加权和）

  4. 作者的方法与现代算法之间的关系（虽然没有应用）


 | 核心思想   | 作者频域方法                            | 现代ISP方法                                             |
 | ------ | --------------------------------- | --------------------------------------------------- |
 | 先重建G通道 | 因为G通道采样率最高，承载了主要的亮度信息，且与R/B通道高频相似 | 一致                                                  |
 | 利用色差   | 在频域中表现为调制到高频的色差信号（R-G, B-G）       | 在空间域中体现为对色差（G-R, G-B）进行插值，而非直接插值R、B。这是所有基于色差方法的共同基础 |
 | 边缘导向插值 | 自适应地选择混叠更小的方向（水平或垂直）进行重建          | 完全一致。HA和LED算法的核心都是计算水平和垂直方向的梯度，沿着边缘方向插值，避免跨越边缘      |


  5. 为什么神经网络的去马赛克表现更好：

     更强的模型：非线性+多层结构，表达能力远超线性模型。

     更大的视野：全局感受野，能够利用远距离的上下文信息，而你猜测的“传统算法只用局部信息”确实是其核心局限。

     更深的特征：自动学习从低级到高级的多层级特征，综合利用它们进行重建。

     更好的训练：端到端联合优化，用海量数据驱动数千万参数的学习。

* 既然RGGB这种排列有缺陷，为什么还在流行？

  1. 三原色基于人类的视觉系统，这样设计符合直觉

  2. 技术惯性，太多设备、算法适配了，很难替代

  3. 从RGGB格式重建彩色的信号算法并不复杂（虽然不是最优）

* 彩色ISP 中CFA 是非常核心的模块，以至于其他模块是放在其前还是后也需要考虑。

  1. 显然放在CFA前能够节省计算量，但误差可能会被CFA放大，甚至出现artifacts

  2. 不过现代ISP的pipeline设计都基本一致，估计已经是衡量效果与成本的最优结果了

# CFA 与噪声

* CFA 与噪声

  1. CFA 改变了噪声分布，因此在CFA过程中降噪是比较好的方案（书中花了一章介绍，但是raw域降噪已经是常规手段了，因此不多展开）

  2. 对于彩色噪声，有可能完全消除；对于亮度噪声，基本不可能完全消除

* 噪声+CFA的频域建模

  1. 频域中图像 = 亮度信息+色差信息+噪声

  2. 如果这三个信号分量在频域完全不重叠会能否完美去噪？

     理论层面：如果“完全不重叠”真的成立，并且有理想滤波器，那么是的，可以完美恢复。

     实际情况：

     “完全不重叠”本身就不成立——亮度信号的频谱是无限宽的。

     理想滤波器不存在——物理可实现滤波器都有过渡带。

     采样带来的频谱复制——这是由离散采样这个行为本身决定的，无法改变。

* 作者提出的去马塞克+降噪算法（原书9章）

  1. 路径一：约束滤波法——去马赛克算法的“灵魂”

     这条路径在现代ISP中体现得最为直接和广泛，它基本奠定了后续所有去马赛克算法的设计框架。

     **在ISP中的体现**：现代ISP中的去马赛克（Demosaic）模块，其核心思想就是“边缘导向”，这正是对“约束滤波”思想的延续。算法通过计算水平/垂直梯度，约束插值方向，避免跨越边缘。你之前提到的LED算法，本质上就是对这一路径的精细化改进。

     **为何能延续**：这种方法计算简单，非常适合硬件流水线实现，是性能和复杂度的最佳平衡点。它不是被“照搬”，而是被“固化”成了ISP的标准模块。

  2. 路径二：缺失数据法——理论贡献大于工程实现

     这条路径在现代ISP中没有直接的硬件实现，但其思想被AI-ISP深度吸收。

     **为何难以硬件实现**：这种方法需要迭代求解，计算量大，且依赖复杂的先验模型，难以用硬件流水线实现高吞吐处理。

     **AI-ISP”中的复活**：这条路径的贝叶斯视角，在AI-ISP中找到了新的生命。例如，最新的研究将图像形成模型和去噪、去马赛克问题建模成一个迭代优化问题，然后用一个展开的（unrolled）神经网络来替代这个迭代求解过程。网络通过学习海量数据，隐式地学到了模型的先验，从而完成联合处理。这可以被看作是“缺失数据法”的数据驱动版本。

  3. 路径三：滤波器组系数估计法——小波域的华丽转身

  这条路径在传统ISP硬件中最少见，但在AI-ISP中却迎来了“第二春”，你的猜测在这里得到了印证。

  **为何传统ISP少见**：

  硬件开销大：二维离散小波变换的计算复杂，对存储和带宽要求高，不利于低功耗设计。

  实时性挑战：难以像传统流水线那样做到逐行处理，需要缓存多行数据进行变换。

  生态不兼容：传统ISP是基于“处理模块”的思维，小波分析属于“变换域处理”，难以无缝插入。

  **“AI-ISP”中的大放异彩：**

  专门的小波硬件：专为小波变换设计的ASIP或FPGA加速器已经被研究出来，证明了其硬件实现的可行性。

  作为神经网络的一部分：小波变换被巧妙地融入到深度学习网络中。例如，AWNet 利用离散小波变换（DWT）来提取RAW图像的高低频信息，在保持计算效率的同时恢复图像细节。

  轻量化设计的利器：利用小波分解的特性，可以设计出轻量化的红外噪声抑制方法，显著降低计算量，提升推理速度。

# 白平衡

白平衡问题：

这是一个典型的病态问题：

已知：最终图像的RGB值。

未知：光源的颜色 + 物体本身的反射率。

从一张图像中分离出这两个因素，本身就是高度不确定的。因此，所有AWB算法都必须基于一些关于“这个世界通常是什么样的”假设。

根本挑战：算法无法区分“一个红色的物体在白色光源下”和“一个白色的物体在红色光源下”。



# 到底是先插值还是先矫正？

本书提出先矫正更好，但是现代ISP基本都是后矫正

在纯数学模型下，"先校正后插值"理论上误差更小。这个结论在数学上正确。

现代ISP中确实有部分"先校正"的影子：

白平衡增益在RAW域应用：在去马赛克之前，先对R/B通道乘以增益系数

绿色不平衡校正（GIC）：在Bayer域校正Gr/Gb的差异

色差恒定假设：去马赛克算法本身利用了"色差信号平滑"的特性，这本质上是在去马赛克过程中隐含了色彩相关性

# 如何从连续的低质量的CFA数据重建高质量的图像

1.1 问题的定义

我们有一个低分辨率（LR）的CFA视频序列 ，目标是重建出一个高分辨率（HR）的彩色视频&#x20;

这个问题的挑战在于，它同时包含了三个子问题：去马赛克、超分辨率、运动估计

1.2 为什么不能分开做？

最直观的思路是两步走：

先对每一帧做去马赛克，得到低分辨率彩色图像序列。

再对彩色图像序列做超分辨率，得到高分辨率彩色视频。

但这种方法效果不好，原因在于：

第一步去马赛克的误差会被第二步超分辨率放大。

分开处理时，两个步骤的模型假设可能不一致，导致误差累积。

第19章的核心贡献：提出了一个统一的物理模型，将这三个问题在一个框架内联合求解。



二、第19章的物理模型

2.1 前向成像模型

![[attachments/single-sensor-imaging-image-2.png]]

这个模型告诉我们：我们观测到的低分辨率马赛克图像，是理想高分辨率图像经过运动、模糊、下采样、CFA采样和噪声污染后的结果。

这个模型的精妙之处在于，它将所有的不完美因素都统一在一个数学框架下。从观测数据 重建就是一个逆向求解问题。



三、第19章的求解方法

本章提出了三种不同复杂度的求解方法，层层递进。



3.1 方法一：联合MAP估计（最完整，但最复杂）

核心思想：建立一个统一的代价函数，包含数据保真项和多个先验约束项，然后通过迭代优化求解。

优点：理论上最完备。

缺点：计算量巨大，需要迭代求解，不适合实时应用。



3.2 方法二：两步法（Fast Implementation） 这里不展开了



3.3 方法三：动态处理（Kalman Filter）

核心思想：对于视频序列，引入卡尔曼滤波进行递归处理。利用上一帧的高分辨率结果，预测当前帧的初始值。并用当前的实际值修正预测模型（这里是卡尔曼滤波的基本操作）

优点：适合实时视频处理，计算效率高。

缺点：卡尔曼滤波的模型假设（线性、高斯）可能与真实情况有偏差。



五、与现代技术的联系

本章技术虽然没有落地，但是在神经网络的图像处理中，其思想被延续（联合建模、物理模型+先验约束）并没有过时，它只是被现代深度学习“内化”了——物理模型被用作网络结构的设计蓝图，先验约束被数据驱动的学习所取代，从而实现更好的性能。
