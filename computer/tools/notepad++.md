这是一个24年9月的笔记，同步到我们新的仓库

**具体快捷键只需要打开菜单栏查看**

| 类型    | 快捷键                 | 作用                   |
| ----- | ------------------- | -------------------- |
| 文件操作  | ctrl+N              | 新建文件                 |
|       | ctrl+alt+s          | 另存为                  |
|       | ctrl+shift+s        | 保存所有                 |
|       | Ctrl+Tab            | 文件标签跳转，跳至下一个打开文件     |
|       | Ctrl+Shift+Tab      | 文件标签跳转，跳至上一个打开       |
|       | Ctrl+W              | 关闭当前文件               |
|       | Ctrl-Shift-T        | 打开之前关闭的文件            |
| 复制、剪切 | ctrl+D              | 复制当前行至下方，或者复制选中区域至其后 |
| 撤销    | ctrl+y              | ctrl+z的逆操作           |
| 调整    | ctrl+T              | 将当前行与上一行交换位置、或撤销     |
|       | ctrl+L              | 删除当前行                |
|       | ctrl+J              | 合并选中的多行              |
|       | ctrl+shift+up       | 当前行/选中行向上移动          |
|       | ctrl+shift+down     | 当前行/选中行向下移动          |
| 注释    | ctrl+k              | 选中区域加注释              |
|       | ctrl+shift+k        | 选中区域取消注释             |
| 大小写转换 | ctrl+u/ctrl+shift+u | 转为小写/大写              |
| 跳转    | ctrl+b              | 跳转到配对的括号             |
| 查看    | Ctrl-Alt-R          | 整个页面文字方向从右到左         |
|       | Ctrl-Alt-L          | 整个页面文字方向从左到右         |
|       | alt+h               | 隐藏行                  |
| 选中    | ctrl+shift+上下左右     | 选择多行                 |
|       | alt+shift+上下左右      | 选中举行区域               |
|       | shift+pageup        |                      |
|       | shift+pagedown      |                      |
|       | shift+home          |                      |
|       | shift+end           |                      |
| 折叠    | alt+数字              | 收缩相应的折叠              |
|       | alt+shift+数数字       | 展开相应的折叠              |
|       | alt+0               | 全折叠                  |
|       | alt+shift+0         | 全展开                  |
| 列编辑   | alt+c               | 打开列编辑对话框             |
|       | 选中+tab              |                      |
|       | 选中+shift+tab        |                      |
